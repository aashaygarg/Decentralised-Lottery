# Decentralised Lottery

1. Users can enter lottery with ETH based on a USD fee
2. An admin will choose when the lottery is over
3. The lottery will select a random winner

## Technologies Used

* Solidity
* Python
* Remix
* Ganache
* Rinkeby
* Brownie
* Py-test
* Openzeppelin
* True Randomness with Chainlink VRF
